For Vigenére Cipher, the key for the encryption is a vector, chosen by the description below.

We first choose a key length, 6 as an example. Then choose a vector of this size whose entries are integers from 0 to 25. Given an example, we have the k = (21, 4, 2, 19, 14, 17). The key often corresponds to a word that is easily remembered. We build the word vector in this example.

The security of the system depends on the fact that neither the keyword nor its length is known.
